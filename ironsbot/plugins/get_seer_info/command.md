# 赛尔号数据查询插件

赛尔号数据查询插件。

- 支持**名称模糊搜索**和 **ID 精确查找**；多个结果时会提示选择。
- 名称搜索时可省略 `·` `-` 等符号，不区分大小写。
- 所有命令优先在消息**开头**触发（部分命令也可放在消息结尾），命令和参数间无需空格。
- 回复他人消息时不会触发命令。

---

## 精灵相关

### 精灵信息

查询精灵基础信息（属性、魂印、技能等）。

| 命令   | 别名                           | 参数        |
| ------ | ------------------------------ | ----------- |
| `精灵` | `查询精灵信息`、`魂印`、`技能` | `<名称/ID>` |

**示例：**

```
精灵雷伊        # 查询名称含"雷伊"的精灵信息
奥菲魂印        # 通过别名"魂印"查询奥菲的信息
圣武技能        # 通过别名"技能"查询圣武的信息
```

---
